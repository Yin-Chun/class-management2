🚀 班級管理系統 - 快速部署步驟

═══════════════════════════════════════════

📁 您需要上傳的檔案：
   ✓ index.html （這就是班級管理系統）

═══════════════════════════════════════════

⚡ 5分鐘快速部署：

1️⃣ 註冊 GitHub
   → 前往 https://github.com/
   → 點擊 Sign up 註冊帳號
   → 記住您的使用者名稱！

2️⃣ 創建儲存庫
   → 登入後點擊右上角 + 號
   → 選擇 New repository
   → 名稱輸入：class-management
   → 選擇 Public（公開）
   → 勾選 Add a README file
   → 點擊 Create repository

3️⃣ 上傳檔案
   → 點擊 Add file → Upload files
   → 拖曳 index.html 到頁面
   → 點擊 Commit changes

4️⃣ 啟用網站
   → 點擊 Settings（設定）
   → 左側選單點擊 Pages
   → Branch 選擇 main
   → Folder 選擇 /root
   → 點擊 Save
   → 等待 1-2 分鐘

5️⃣ 取得網址
   您的網址：
   https://您的使用者名稱.github.io/class-management/

═══════════════════════════════════════════

💡 提示：
• 將網址加入書籤，方便每日使用
• 手機也可以訪問相同網址
• 所有資料存在瀏覽器，非常安全
