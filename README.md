# 班級管理小幫手 - 部署指南

## 🌐 將系統部署到網路上（GitHub Pages）

按照以下步驟，您就可以獲得一個真實的網址，隨時隨地訪問您的班級管理系統！

---

## 📋 部署步驟

### 步驟 1：註冊 GitHub 帳號

1. 前往 [GitHub 官網](https://github.com/)
2. 點擊右上角的 **Sign up**（註冊）
3. 填寫您的：
   - 電子郵件
   - 密碼
   - 使用者名稱（這個很重要，會出現在網址中）
4. 完成註冊並驗證電子郵件

---

### 步驟 2：創建新的儲存庫（Repository）

1. 登入 GitHub 後，點擊右上角的 **+** 號
2. 選擇 **New repository**
3. 填寫資料：
   - **Repository name**（儲存庫名稱）：輸入 `class-management`（或任何您喜歡的名稱）
   - **Description**（說明）：可選填，例如「國小班級管理系統」
   - 選擇 **Public**（公開）
   - ✅ 勾選 **Add a README file**
4. 點擊 **Create repository**（創建儲存庫）

---

### 步驟 3：上傳檔案

1. 在您剛創建的儲存庫頁面中，點擊 **Add file** → **Upload files**
2. 將以下檔案拖曳到上傳區域：
   - `index.html`（這就是您的班級管理系統）
3. 在下方填寫提交訊息：`新增班級管理系統`
4. 點擊 **Commit changes**（提交變更）

---

### 步驟 4：啟用 GitHub Pages

1. 在您的儲存庫頁面中，點擊上方的 **Settings**（設定）
2. 在左側選單中找到 **Pages**
3. 在 **Source**（來源）部分：
   - **Branch**（分支）：選擇 `main`
   - **Folder**（資料夾）：選擇 `/root`
4. 點擊 **Save**（儲存）
5. 等待約 1-2 分鐘，頁面會顯示您的網址

---

### 步驟 5：訪問您的網站

您的網址格式會是：
```
https://您的使用者名稱.github.io/class-management/
```

例如，如果您的使用者名稱是 `teacher-wang`，儲存庫名稱是 `class-management`，那麼網址就是：
```
https://teacher-wang.github.io/class-management/
```

🎉 完成！現在您可以在任何有網路的地方使用這個網址訪問您的班級管理系統了！

## 💡 使用建議

### 建立書籤
1. 在瀏覽器中打開您的網站
2. 按 `Ctrl/Cmd + D` 將網址加入書籤
3. 以後直接點擊書籤即可快速訪問

### 手機訪問
- 您的手機也可以訪問相同的網址
- 建議將網址儲存到手機主畫面（類似 App 的使用體驗）

### 資料保存
- 所有資料（學生名單、提醒、聯絡簿等）都存在瀏覽器中
- 只要使用同一台電腦的同一個瀏覽器，資料會自動保存
- 建議定期截圖或手動備份重要資料

## ❓ 常見問題

**Q: 如果我修改了 HTML 檔案，如何更新網站？**
A: 只要回到 GitHub 儲存庫，重新上傳新的 `index.html` 檔案即可。

**Q: 資料會存在雲端嗎？**
A: 不會。所有資料存在您的瀏覽器本地。網站只是提供程式碼，不會收集或儲存您的資料。

**Q: 需要付費嗎？**
A: 完全免費！GitHub Pages 是免費的服務。

**Q: 其他人可以看到我的網站嗎？**
A: 可以。任何人只要知道網址都能訪問。但他們看到的是空白的系統，看不到您輸入的資料（因為資料存在您自己的瀏覽器中）。

**Q: 如果想要只有自己能訪問怎麼辦？**
A: 建議使用本地檔案方式，或者將儲存庫設為 Private（但 GitHub Pages 對私人儲存庫有使用限制）。
